Stand: 1.2.2020

Temperaturlogger mit Thingspeak-Library.\

Folgende Librarys werden benötigt:
 *  Für ESP12, Node MCU 1.0 Auswählen\
 *  Adafruit ESP8266 by Adafruit\
 *  Adafruit Unified Sensor by Adafruit\
 *  DHT Sensor Library by Adafruid\
